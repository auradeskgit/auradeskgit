## ▓ PROJECTS

<div align="center">

### 🟪 GitCube
**find repositories with AI**

Search GitHub in plain English — GitCube turns a natural-language
query into the right search, then hands back AI-summarized results
so you don't have to dig through repos one by one.

`Vue` `SVG` `localStorage` `AI-powered search`

[![Live Demo](https://img.shields.io/badge/▶_LIVE_DEMO-A855F7?style=for-the-badge&logoColor=white)](#)

<br/>

### 🟪 Candlewick
**live markets + AI desk**

A real-time market dashboard for stocks and crypto, streaming live
price data with an integrated AI desk for on-demand analysis of any
asset or timeframe.

`WebSocket` `Finnhub API` `Real-time data` `AI analysis`

[![Live Demo](https://img.shields.io/badge/▶_LIVE_DEMO-A855F7?style=for-the-badge&logoColor=white)](#)

<br/>

### 🟪 AuraDesk
**premium OS dashboard**

A draggable, widget-based desktop-style dashboard in the browser —
drop in widgets, arrange your own layout, and keep it all saved
locally.

`Canvas` `Chart.js` `WebSocket` `localStorage` `Widget engine`

[![Live Demo](https://img.shields.io/badge/▶_LIVE_DEMO-A855F7?style=for-the-badge&logoColor=white)](#)
